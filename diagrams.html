flowchart TD

PDF/DOCX/TXT]

    Extract --> Chunk
    Chunk --> Embed
    Embed --> Store
    Store --> Index

    Upload -.-> |Async| Celery[Celery Worker]
    Celery --> Extract

7. Authentication Flow (Sequence)

sequenceDiagram
    participant User
    participant FE as Frontend
    participant API as FastAPI
    participant Redis as Redis

    User->>FE: Login credentials
    FE->>API: POST /auth/login
    API->>API: Validate credentials
    API->>API: Generate JWT tokens
    API->>Redis: Store refresh token
    API-->>FE: access_token + refresh_token
    FE-->>User: Login success

    Note over User,FE: Subsequent requests with JWT
    FE->>API: Request + access_token
    API-->>FE: Response
    API->>API: Check token expiry
