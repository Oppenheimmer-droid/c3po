# C3PO - ReDrive Edu

AI-powered educational platform for automatic quiz generation.

## Quick Start

### Frontend (Vercel)
- **Live**: https://c3po-lime.vercel.app
- **Root**: `frontend/`

### Backend (Railway)
- **API**: https://c3po-production-0c24.up.railway.app
- **Root**: `backend/`

## Development

```bash
# Backend
cd backend
docker-compose up

# Frontend
cd frontend
npm install && npm run dev
```
